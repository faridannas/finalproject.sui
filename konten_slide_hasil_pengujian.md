# Konten Slide Hasil Pengujian (Detail Persentase)

Jika kamu menggunakan diagram lingkaran (Pie Chart) dengan 5 bagian seperti di gambar, berikut adalah pembagian yang logis untuk sistem Seblak Umi. Setiap bagian memiliki bobot **20%** sehingga totalnya menjadi **100%**.

---

## Pembagian Item Diagram (5 Bagian x 20%)

Ganti label "Item 1" s/d "Item 5" dengan teks berikut:

1.  **Item 1: Akun & Keamanan (20%)**
    *   *Meliputi:* Login, Register, Logout, dan proteksi halaman Admin (Middleware).

2.  **Item 2: Produk & Stok (20%)**
    *   *Meliputi:* Tambah/Edit/Hapus Menu Seblak, Kategori, dan pengurangan stok otomatis.

3.  **Item 3: Transaksi Pesanan (20%)**
    *   *Meliputi:* Masuk keranjang (Cart), Checkout, dan penyimpanan data pesanan.

4.  **Item 4: Pembayaran Digital (20%)**
    *   *Meliputi:* Integrasi API Midtrans (QRIS/Bank Transfer) dan update status bayar otomatis.

5.  **Item 5: Laporan & Admin (20%)**
    *   *Meliputi:* Dashboard Admin, Export Laporan Excel/PDF, dan Manajemen Testimoni.

---

**Kesimpulan:**
Total 5 Bagian x 20% = **100% Sistem Berjalan Valid.**
